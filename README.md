# Sentinel Demo

A minimal React + TypeScript demo app integrating `@testpress/sentinel-core` and `@testpress/sentinel-ui`.

## Setup

```bash
pnpm install
```

## Run

```bash
pnpm dev
```

Open the URL printed by Vite (default `http://localhost:5173`).

## Usage

1. Enter `orgCode`, `sessionId`, and `token` values from a valid Sentinel session.
2. Click **Launch Sentinel** — the overlay is mounted and the setup flow starts automatically.
3. Use **Reset** to unmount the overlay and return to idle.
